* COMANDOS GITHUB
> - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

Nota: 
Recuerda agregar una credencial web de GitHub
en tu PC, o bien quitar las credenciales de otro usuario y 
agregar tus credenciales, de lo contraio no podras subir 
tu proyecto a GitHub.

> - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
Pasos previos

1 Detener el servidor(terminal):

Control+c

2 Especificar dependencias mediante un archivo de requisitos txt(terminal)
en sistema(raiz)>

pip freeze
pip freeze > requirements.txt

3 Desactivo el entorno virtual(terminal):

deactivate

4 Exportar base de datos de SqlYog a Sistema

5 Crear un archivo README.md con la
documentacion del proyecto.

6 Crear un archivo .gitignore en sistema(raiz)>
y poner los siguientes archivos a ignorar:

__pycache__
env

7 Guardar todos los cambios del proyecto.

8 Crear repositorio en GitHub.

> - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
Subir repositorio(terminal):

git init
git add .
git commit -m "Subiendo proyecto"
git branch -M main
git remote add origin https://github.com/YobaniChicunque/sistema.git
git push -u origin main

> - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
Agregar cambios GitHub(terminal)

git add .
git commit -m "Nuevos cambios"
git push

> - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
Bajar cambios GitHub(terminal)

git pull

> - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
Clonar proyecto GitHub(terminal)

git clone https://github.com/YobaniChicunque/sistema.git

> - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
Correr proyecto clonado(terminal)

1 Abrir la base de datos del proyecto
desde sqlyog

2 Crear el entorno virtual y activarlo(terminal):

python -m venv env
env\scripts\activate

3 Instalar las dependencias del proyecto(terminal):

pip install -r requirements.txt

4 Correr el servidor(terminal):

Proyectos Flask

python app.py 

Proyectos Django

py manage.py runserver
